event 2018-01-08
Persona 能引入協作會議嗎？健保案的創新嘗試

2018 年 1 月 5 日，公共數位創新空間（PDIS）於空總創新基地舉辦「全民健保，長年旅居國外、退保期間過長者，不得成為納保對象」的協作會議。經過一整天的討論，與會者感受上主要認為可依照旅外國人當地醫療資源的取得容易與否，或是出國原因作為可否停保之切分標準；也有許多人認為應當全面廢除停保。這些意見在收集之後，都會在1月8日之政務會議上向行政院長與相關政務委員報告，作為後續政策研擬參考。

event 2017-12-14
粗略共識到政策宣示有多遠的距離？澎湖南方四島案記者會後觀察

在11月3日召開澎湖案後，政委唐鳳以會議所得到的相關資料向行政院長賴清德報告，並將相關資料供有關的部會首長參考。賴院長希望以記者會的形式，向大眾發佈澎湖案的成果。因此，在澎湖案協作會議上所談出來的建議，有了進一步落實的空間。

event 2017-12-02
鞭刑案協作會議聚焦反酒駕 與會者望能有具體成果

會議上，經過大家的討論，順利討論出了針對酒駕的各種可能解法。提案人及附議人希望會議未來能對政府產生具體影響，唐鳳則回應，將會在下週一政務會議上向行政院長及相關首長進行報告。

event 2017-11-25
「不打假球」，時區案協作會議正反雙方均買單

附議人坦承，他認為大部分支持時區者，最主要仍是希望能透過時區調整，讓世界上其他國家能了解台灣與中國大陸的差異；而反對案的附議人雖認為時區調整無法達到此目的，但亦認同台灣應與中國大陸做出差異。最終議題討論如何讓世界看見台灣與中國大陸的不同，雙方均對此感到滿意。

event 2017-11-04
南方四島立即列禁漁區？數位落差致漁民誤會

2017年11月3日舉辦了開放政府聯絡人協作會議，就「東西吉廊道劃設禁漁區」議題做討論。經過會場討論，當地漁民與潛水愛好者均同意永續漁業的重要，不過後續該如何進行，大家仍有不同的想像。

event 2017-10-26
協作會議可以處理勞動議題嗎？七天假協作會議後記

勞動議題一直是社會高度關心的議題。工鬥團體擔心勞基法修法造成勞工權益受損，因此提案希望透過增訂國定假日的方式降低勞工工作日數。此次協作會議在10月25日召開之後，將會彙整議題相關脈絡，供政府後續政策推動參考。

event 2017-10-19
雙胞胎協作會議，讓政府與民眾同步資訊

長期以來，雙胞胎家長由於一次照顧兩個同齡小孩，造成托育方面的沈重壓力。提案人雙胞胎協會的成員與政府相關部會一同參與協作會議，討論雙胞胎議題背後相關的議題脈絡，以及在現行法規框架下可能的解決方向，提供政府部會後續在政策規劃時作為參考。
